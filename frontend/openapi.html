Open API — 가상 주식 매매

v1

외부 시스템(봇, 트레이딩 프로그램 등)이 EDUMGT 모의투자 플랫폼의 주식 계좌를 API Key로 인증해 조회·매매할 수 있는 REST API입니다. 웹에서 로그인해 매매한 것과 동일한 예수금(KRW 캐시)·보유종목을 그대로 공유합니다.

인증 (Authentication)

모든 요청은 Authorization 헤더에 발급받은 API 키를 Bearer 토큰 형식으로 담아 전송합니다. 키는 발급 시 1회만 원문으로 표시되며, 서버에는 해시값만 저장됩니다. 유출이 의심되면 즉시 폐기(revoke) 후 재발급하세요. 

Authorization: Bearer eduapi_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Base URL: https://<서비스 도메인>/openapi/v1  |  요청 제한: 키당 분당 60회 (초과 시 429)

엔드포인트

MethodPath설명
GET/stocks매매 가능 종목 목록
GET/quote/<symbol>실시간(캐시 60초) 시세 조회
GET/account예수금(cash), 총자산, 손익률
GET/positions보유 종목 목록 및 평가손익
POST/orders주문 실행 (시장가 매수/매도)
GET/orders?limit=50주문/체결 내역 조회

사용 예시

1. 계좌 조회

curl https://<서비스 도메인>/openapi/v1/account \
  -H "Authorization: Bearer eduapi_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

# 응답
{ "cash": 9852296, "totalAsset": 10000000, "totalPnlRate": 0.0 }

2. 삼성전자(005930) 1주 시장가 매수

curl -X POST https://<서비스 도메인>/openapi/v1/orders \
  -H "Authorization: Bearer eduapi_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{"symbol": "005930", "side": "BUY", "quantity": 1}'

# 응답
{ "status": "ok", "symbol": "005930", "side": "BUY", "quantity": 1, "price": 73852, "amount": 73852 }

3. 보유 종목 조회

curl https://<서비스 도메인>/openapi/v1/positions \
  -H "Authorization: Bearer eduapi_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

# 응답
{ "positions": [
  { "symbol": "005930", "name": "삼성전자", "quantity": 3, "avgPrice": 73852,
    "currentPrice": 73852, "evalAmount": 221556, "pnl": 0 }
] }
